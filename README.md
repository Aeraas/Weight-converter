# Weight Converter

A simple Java console application that converts:
- Pounds (lbs) to Kilograms (kg)
- Kilograms (kg) to Pounds (lbs)

## Features
- Menu-based interface
- Input Correction
- Reusable conversion methods
- Clean and readable code structure

## How to Run
1. Compile the program:
   javac WeightConverter.java

2. Run the program:
   java WeightConverter

## Example
Menu:
1. Convert pounds (lbs) to kilograms (kg)
2. Convert kilograms (kg) to pounds (lbs)
3. Exit
Choose an option: 1
Enter weight in lbs: 150
150.00 lbs = 68.04 kg
